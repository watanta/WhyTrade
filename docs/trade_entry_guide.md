# WhyTrade 新規ポジション作成チェックリスト & 入力ガイド

このドキュメントは、新規ポジション（トレード）を作成する際に確認すべき項目を**チェックリスト形式**でまとめたものです。
エントリー前にこのリストを一通り確認することで、見落としを防ぎ、より根拠のあるトレードを行うことができます。

## 1. 基本情報 (Basic Information)
*   [ ] **銘柄コード**: 正しい銘柄か？（類似コードとの間違いはないか）
*   [ ] **売買区分**: 「買い」と「売り（空売り）」を間違えていないか？
*   [ ] **数量**: 意図したロット数か？（100株と1000株を間違えていないか）

---

## 2. 理由・根拠 (Why?)

### 市場環境 (Market Environment)
市場全体の地合いを確認しましたか？

<details>
<summary>📊 データの見方・参照先</summary>

*   **指標チェック**: [TradingView](https://jp.tradingview.com/) や [Investing.com](https://jp.investing.com/)
    *   **金利**: `US10Y` (米国債10年利回り) - 上昇中は株安圧力になりやすい。
    *   **VIX**: `VIX` - 20以下は安定、30以上はパニック。
    *   **為替**: `USDJPY` - 円安は輸出株(自動車など)にプラス、輸入株にマイナス。
*   **イベント**: [Yahoo!ファイナンス 経済指標カレンダー](https://info.finance.yahoo.co.jp/fx/marketcalendar/)
    *   ★マークの多い重要指標（CPI, 雇用統計, FOMC）の発表直前はエントリーを避ける。
</details>

*   [ ] **マクロ**: 重要な経済指標の発表直前ではないか？ 金利・為替に急激な変動はないか？
*   [ ] **トレンド**: 主要指数（日経平均、NYダウ）は自分のトレード方向と合致しているか？
*   [ ] **セクター**: 該当セクターに資金は来ているか？（[セクター別騰落率](https://nikkei225jp.com/nikkei/sector/)等で確認）
*   [ ] **VIX**: 恐怖指数が高すぎないか？（パニック相場ではないか）

### テクニカル分析 (Technical Analysis)
チャートに明確な根拠はありますか？

<details>
<summary>📈 データの見方・参照先</summary>

*   **ツール**: 証券会社のツール (楽天 MarketSpeed II, 松井証券 等) または [TradingView](https://jp.tradingview.com/)
*   **チェック手順**:
    1.  **月足・週足**を見る → 大きなトレンド（上か下か）を把握。
    2.  **日足**を見る → 現在の位置（移動平均線との関係、RSI）を確認。
    3.  **執行足（5分足等）**を見る → 具体的なエントリータイミングを探る。
</details>

*   [ ] **トレンド定義**: ダウ理論でトレンドは発生しているか？（高値・安値の切り上げ/切り下げ）
*   [ ] **上位足**: 週足・月足などの上位足でも優位性があるか？（順張りか、根拠のある逆張りか）
*   [ ] **ライン**: 重要なサポート・レジスタンスラインを背にしているか？
*   [ ] **パターン**: ダブルボトム、三角持ち合いなど認識できるパターンがあるか？
*   [ ] **出来高**: ブレイクアウト等のアクションに伴って出来高は増えているか？
*   [ ] **インジケーター**: オシレーター（RSI等）で過熱感をチェックしたか？ 逆行現象（ダイバージェンス）はないか？

### ファンダメンタル分析 (Fundamental Analysis)
企業の価値や材料を確認しましたか？（※短期トレードの場合は重要度低）

<details>
<summary>🏢 データの見方・参照先</summary>

*   **日本株**: [株探 (Kabutan)](https://kabutan.jp/)
    *   **決算**: トップページの「決算速報」や、銘柄ページの「決算」タブで **「3期比較」** を確認。売上・利益が伸びているか？
    *   **進捗**: 会社予想に対する進捗率は順調か？
*   **米国株**: [Seeking Alpha](https://seekingalpha.com/) や [Yahoo Finance](https://finance.yahoo.com/)
    *   **Earnings**: 予想 (Estimate) に対して結果 (Actual) が上回ったか (Beat) を確認。
</details>

*   [ ] **決算**: 直近の決算内容は良かったか？（サプライズ有無） 次回決算日はいつか？（持ち越すリスク確認）
*   [ ] **バリュエーション**: PER/PBR的に割高ではないか？ または正当化できる成長性があるか？
*   [ ] **テーマ**: 今の市場で注目されているテーマ（例: AI、インバウンド）に関連しているか？
*   [ ] **カタリスト/材料**: 株価を動かす具体的なニュースや期待材料はあるか？（[適時開示情報検索](https://www.release.tdnet.info/inbs/I_main_00.html)等）

### 確信度 (Confidence Level)
*   [ ] そのトレードは、自分の「勝ちパターン」に合致しているか？
    *   (5: 鉄板パターン, 3: 通常ルール通り, 1: 実験的・打診)

## 3. エントリー計画 (When & How?)

### エントリータイミング

<details>
<summary>⏱ データの見方・参照先</summary>

*   **板情報 (Order Book)**: 証券会社のツールで「板」を見る。
    *   **見せ板**: 異常に大きな注文がないか？
    *   **買い圧/売り圧**: 歩み値 (Time & Sales) で、どちらの注文が多く約定しているか？
</details>

*   [ ] **トリガー**: エントリーの「具体的な引き金」は引かれたか？（見切り発車ではないか）
*   [ ] **待機**: ちゃんと「押し目」や「ブレイク確定」まで待てたか？
*   [ ] **時間帯**: 寄り付き直後の乱高下や、引け間際のリスクを考慮したか？

### 出口戦略 (Exit Strategy)
エントリー前に出口は決まっていますか？
*   [ ] **損切り (Stop)**: **「根拠が崩れる価格」**に逆指値を置いているか？（自分の都合で決めていないか）
*   [ ] **利確 (Target)**: 節目や計算値に基づいた目標価格を設定したか？
*   [ ] **リスクリワード**: (目標利益 ÷ 許容損失) は **2.0以上** あるか？
*   [ ] **保有期間**: デイトレか持ち越しか、事前に決めているか？

### 資金管理 (Position Sizing)
*   [ ] **最大損失額**: もし損切りになった場合、その損失額は許容範囲内（例: 資金の1%〜2%）か？
*   [ ] **ロット調整**: ボラティリティの高さに応じてロットを調整したか？
*   [ ] **余力**: ナンピンや追撃をするための余力は考慮されているか？

（フォームの「トレードシナリオ・総合判断」欄への記入イメージ）

> 全体相場は調整中だが、半導体セクターへ資金循環が見られる（市場）。
> この銘柄は好決算後に調整していたが、75日線で下ヒゲをつけて反発し、本日出来高増で直近高値をブレイクした（テクニカル・ファンダ）。
> 下値は限定的と判断し、前回高値を目標にエントリー。損切りは75日線割れに設定しリスクリワード1:3を確保。
